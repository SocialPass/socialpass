# socialpass backend
socialpass backend
main application is `backend/root`


## Local Development
# Requirements
Python3+ and local Postgres titled 'v3db'

# Initial Setup
`git clone git@github.com:nftylabs/v3.git`
`python3 -m venv venv`
`source venv/bin/activate`
`pip3 install -r requirements/local.txt`


# setup pre-commit hooks
pre-commit install
